# Highlight Bar Quick Start

Use this if you want to get running fast.

## 1) Download

- Latest release: `https://github.com/25mattm/Highlighter/releases/latest`
- macOS: `HighlightBar-macos.zip`
- Windows: `HighlightBar-windows-x64.zip`

## 2) Install And Open

### macOS

1. Download and unzip `HighlightBar-macos.zip`.
2. Drag `HighlightBar.app` to `Applications`.
3. Open `HighlightBar.app`.

If blocked: `System Settings -> Privacy & Security -> Open Anyway`.

### Windows

1. Download and unzip `HighlightBar-windows-x64.zip`.
2. Run `HighlightBar.Windows.exe`.

If SmartScreen appears: `More info -> Run anyway`.

## 3) Use It

1. Find the app icon:
- macOS: `HB` in menu bar
- Windows: tray icon near clock
2. Adjust:
- height
- transparency
- color
3. Quit from menu: `Quit Highlight Bar`
